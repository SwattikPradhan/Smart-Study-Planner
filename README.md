
# 📚 Smart Study Planner (AIML Project)

## 🚀 Overview
Smart Study Planner is an AI-based web application that predicts student productivity based on study habits and suggests optimal schedules.

## 🧠 Features
- Predict productivity score
- Analyze study patterns
- Suggest improvements

## ⚙️ Tech Stack
- Python
- Scikit-learn
- Streamlit

## ▶️ How to Run

1. Clone repo:
git clone https://github.com/SwattikPradhan/smart-study-planner.git

2. Install dependencies:
pip install -r requirements.txt

3. Run app:
streamlit run app.py

## 📊 Model
Linear Regression model trained on study habit data.
