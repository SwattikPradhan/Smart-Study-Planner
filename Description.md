# 📚 Smart Study Planner

A machine learning-powered web application that predicts a user's productivity score based on daily habits. This tool helps students and professionals optimize their schedules by analyzing the relationship between study time, sleep, and breaks.

---

## 🚀 Features

* **Productivity Prediction:** Uses a **Linear Regression** model to estimate a productivity score (1–10).
* **Interactive UI:** Built with **Streamlit**, featuring intuitive sliders for real-time data input.
* **Smart Feedback:** Provides personalized encouragement based on the predicted score.
* **Data-Driven:** Translates habit patterns into actionable performance insights.

---

## 🛠️ Tech Stack

* **Frontend:** [Streamlit](https://streamlit.io/)
* **Machine Learning:** Scikit-learn (Linear Regression)
* **Data Processing:** Pandas & NumPy
* **Model Serialization:** Pickle

## 📊 How It Works

The **Smart Study Planner** bridges the gap between raw habit data and productivity insights using a supervised Machine Learning approach.

### 1. Data Collection & Preprocessing
The model is trained on a synthetic dataset (found in `model.py`) that maps three key features to a target productivity score:
* **Study Hours:** Total time spent focused on academic tasks.
* **Sleep Hours:** The amount of rest achieved the previous night.
* **Break Time:** The duration of intervals taken to prevent burnout.

### 2. The Machine Learning Model
We utilize **Linear Regression**, a fundamental predictive algorithm. It calculates the "line of best fit" through the data points to understand how each hour of sleep or study impacts the final score.

The mathematical representation used by the app is:
$$Productivity = \beta_0 + \beta_1(\text{Study}) + \beta_2(\text{Sleep}) + \beta_3(\text{Breaks})$$

*Where $\beta$ represents the weights (coefficients) the model learns during training.*

### 3. Model Deployment (The Pipeline)
* **Training:** `model.py` fits the Linear Regression model to the data and "freezes" it into a `model.pkl` file using the **Pickle** library.
* **Interface:** `app.py` uses **Streamlit** to create a web-based dashboard.
* **Prediction:** When a user moves the sliders, the app loads `model.pkl`, passes the slider values into the model, and displays the result instantly.

### 4. Interpretation Logic
The app categorizes the numerical output into three visual feedback zones:
| Score Range | Category | Actionable Advice |
| :--- | :--- | :--- |
| **8.0 - 10.0** | 🔥 Peak Performance | You have found the "Goldilocks" zone of habits. |
| **5.0 - 7.9** | 👍 Good / Average | Minor adjustments to sleep or breaks could boost results. |
| **Below 5.0** | ⚠️ Needs Adjustment | Your current balance may lead to fatigue or inefficiency. |
